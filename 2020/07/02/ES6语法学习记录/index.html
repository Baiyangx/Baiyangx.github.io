3.简单的搜索关键字

3.1 老方法indexOf

  • indexOf方法将从头到尾地检索字符串 stringObject，看它是否含有子串 searchvalue。开始检索的位置在字符串的 fromindex 处或字符串的开头（没有指定 fromindex 时）。如果找到一个 searchvalue，则返回 searchvalue 的第一次出现的位置。stringObject 中的字符位置是从 0 开始的。

  • 如果要检索的字符串值没有出现，则该方法返回 -1。

  • 如果searchvalue为空字符串‘’，则返回0，在本例中表现为，若搜索关键字为空，(this.list[i].name.indexOf(this.keywords) != -1)判断为真，每项都push进result数组，即返回整个list。

3.2 include

在判断是否包含关键字的逻辑里使用includes方法

该方法根据是否包含子字符串返回布尔值

3.3 filter

  • newList=oldList.filter(callback)传入回调函数，通过回调函数返回的布尔值判断每一项是否满足条件，再把满足条件的项组成一个新数组返回。
  • 注意回调函数中存在this指向的问题。使用箭头函数时回调函数中的this指向与外层一致，即为vue
  • 注意this指向是根据函数运行时的'最近'调用者决定的，在本例中，如果不使用箭头函数，在filter方法中的回调函数中使用this.keywords,最近调用者是一个数组，this指向为windows。

this指向问题&箭头函数格式问题 参考： [ES6箭头函数中this的理解][https://blog.csdn.net/weixin_41993254/article/details/106630910]
